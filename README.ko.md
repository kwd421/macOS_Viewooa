# Viewooa

[English README](README.md)

Viewooa는 네이티브 macOS 포토뷰어입니다. Finder급 비주얼 파일 브라우저인 자체파인더와 함께 발전하며, 사진 보기와 파일 탐색을 서로 독립적인 앱처럼 유지하면서도 자연스럽게 연결하는 것을 목표로 합니다.

![Viewooa 뷰어](docs/assets/viewooa-viewer.png)

## 할 수 있는 일

- 이미지, 폴더, PDF, 지원되는 카메라 RAW 파일을 열 수 있습니다.
- 이전/다음 버튼, 방향키, 트랙패드 제스처, 마우스 휠로 폴더 안 이미지를 넘겨볼 수 있습니다.
- 단일 페이지, L-R, R-L, 표지 모드, 세로 웹툰형 보기로 이미지를 볼 수 있습니다.
- 전체 맞춤, 가로 맞춤, 세로 맞춤으로 이미지를 맞추고, 1배율 또는 원하는 배율로 확대/축소할 수 있습니다.
- 핀치 제스처, Command + 마우스 휠, 툴바 버튼, 더블클릭으로 확대/축소할 수 있습니다.
- 이미지가 창보다 클 때는 드래그해서 원하는 부분으로 이동할 수 있습니다.
- 이미지를 회전하고, 메타데이터를 확인하고, 가벼운 후처리 옵션을 켜고 끌 수 있습니다.
- 슬라이드쇼를 재생할 수 있으며, 세로 웹툰형 보기에서는 세로 스크롤 재생도 사용할 수 있습니다.
- 뷰어 안에서 자체파인더를 열어 다른 파일이나 폴더를 바로 선택할 수 있습니다.

## 자체파인더

자체파인더는 폴더와 파일을 익숙한 macOS 방식으로 열 수 있는 프로젝트의 Finder 스타일 브라우저입니다.

![자체파인더 브라우저](docs/assets/viewooa-browser.png)

자체파인더에서 지원하는 기능:

- 즐겨찾기와 위치를 사이드바에서 빠르게 이동할 수 있습니다.
- 아이콘 보기와 리스트 보기를 사용할 수 있습니다.
- 뒤로/앞으로 이동을 지원합니다.
- 검색 아이콘을 누르면 확장되는 검색 필드를 사용할 수 있습니다.
- 썸네일 크기를 조절할 수 있습니다.
- 선택, 전체 선택, Shift 범위 선택, 빈 공간 드래그 선택을 지원합니다.
- 선택한 파일까지 이어지는 Finder 스타일 아이콘 경로 표시를 지원합니다.
- 자체파인더에서 폴더를 탐색하고, 단독 브라우저 앱에서는 파일을 직접 열 수 있습니다.

## 뷰어 컨트롤

뷰어는 이미지가 중심이 되도록 플로팅 글래스 스타일 컨트롤을 사용합니다.

- 상단바: 정보, 보기 방식, 맞춤 방식, 슬라이드쇼, 툴바 고정.
- 하단바: 자체파인더 열기, 축소, 1배율, 확대, 회전, 이전, 다음, 고정.
- 숨겨진 툴바는 마우스를 가까이 가져가면 나타납니다.
- 고정된 툴바는 계속 표시되며 마지막 고정 상태를 기억합니다.

## 지원 파일

Viewooa는 일반적인 이미지 감상 흐름에 맞춰져 있습니다.

- macOS에서 지원하는 표준 이미지 형식: JPEG, PNG, HEIC, TIFF, GIF, BMP, WebP 등.
- 직접 선택해서 여는 PDF 파일.
- macOS Uniform Type Identifiers에서 지원하는 카메라 RAW 형식. Canon, Nikon, Sony, Fujifilm, Panasonic, Olympus / OM System, Leica, Pentax 등 주요 카메라 제조사의 RAW 파일을 포함합니다.

PDF 파일은 직접 선택했을 때만 열립니다. 일반 폴더 이미지 탐색 중에는 PDF가 이미지 목록에 섞이지 않습니다.

## 실행

더블클릭:

```text
Open Viewooa.command
```

또는 터미널에서 실행:

```bash
./script/build_and_run.sh
```

## 참고

Viewooa와 자체파인더는 계속 발전 중입니다. 현재는 부드러운 macOS 포토뷰어, Finder급 브라우저, 숨겨진 삭제나 이동 없이 안전하게 파일을 탐색하는 경험에 집중하고 있습니다.
